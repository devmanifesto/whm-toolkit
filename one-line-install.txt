# Instalación de una línea del WHM Toolkit desde GitHub

## 🚀 Instalación (detecta y actualiza instalaciones existentes)

### Opción 1: Con wget
wget -qO- https://raw.githubusercontent.com/devmanifesto/whm-toolkit/main/install_from_github.sh | bash

### Opción 2: Con curl
curl -sSL https://raw.githubusercontent.com/devmanifesto/whm-toolkit/main/install_from_github.sh | bash

### Opción 3: Descarga y ejecuta
wget https://raw.githubusercontent.com/devmanifesto/whm-toolkit/main/install_from_github.sh && chmod +x install_from_github.sh && ./install_from_github.sh

## 🗑️ Desinstalación

### Opción 1: Desinstalación automática (recomendada para pipes)
wget -qO- https://raw.githubusercontent.com/devmanifesto/whm-toolkit/main/uninstall_auto.sh | bash

### Opción 2: Desinstalación con confirmación (interactiva)
wget -qO- https://raw.githubusercontent.com/devmanifesto/whm-toolkit/main/uninstall_from_github.sh | bash

### Opción 3: Con curl
curl -sSL https://raw.githubusercontent.com/devmanifesto/whm-toolkit/main/uninstall_auto.sh | bash

## ✨ Características del instalador:
- ✅ Detecta instalaciones existentes
- ✅ Crea respaldos automáticos
- ✅ Actualiza instalaciones previas
- ✅ Verifica sintaxis y permisos
- ✅ Prueba automática del plugin
- ✅ Instrucciones claras de uso
