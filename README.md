# WHM Toolkit v4.0 - Plugin Oficial

Plugin completamente funcional para WHM (Web Host Manager) basado en la documentación oficial de cPanel/WHM.

## 🚀 **Instalación en Una Línea**

```bash
curl -sSL https://raw.githubusercontent.com/devmanifesto/whm-toolkit/main/install-whm-official.sh | bash
```

## 🧹 **Limpieza Completa**

```bash
curl -sSL https://raw.githubusercontent.com/devmanifesto/whm-toolkit/main/cleanup.sh | bash
```

## ✅ **Características**

- ✅ **Estructura Oficial**: Basada en documentación oficial de WHM
- ✅ **Instalación Automática**: Un comando desde GitHub
- ✅ **Interfaz Moderna**: Diseño responsivo y profesional
- ✅ **Hello World Funcional**: Prueba básica con información del sistema
- ✅ **Sistema de Diagnóstico**: Herramientas integradas
- ✅ **Compatibilidad Total**: Funciona con todas las versiones de WHM

## 📁 **Estructura del Plugin**

```
/usr/local/cpanel/whostmgr/addonfeatures/whm-toolkit/whm-toolkit.cgi
/var/cpanel/apps/whm_toolkit.conf
```

## 🌐 **Acceso**

- **Menú WHM**: Plugins → WHM_Toolkit
- **URL Directa**: `https://tu-servidor:2087/addonfeatures/whm-toolkit/whm-toolkit.cgi`
